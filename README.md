# zypython
Haxe-python的通用使用库

# 版本更新
0.0.5:
- 新增：新增命令行`cp`支持，例子：`haxelib run zypython cp [file] [tofile]`。

0.0.4:
- 新增：新增Zip支持。

0.0.3:
- [改进]FileUtils.copyDic新增了拷贝文件夹时，可忽略指定类型/文件
- [新增]FileUtils.removeNoneDic新增了删除空文件夹API

0.0.2:
- [改进]新增FTP功能

0.0.1:
- 新增FileUtils，安全删除文件夹，拷贝文件夹
- 新增FPT常规API
- 新增MD5计算文件
- 新增GetPass
- 新增HttpDownload，用于下载文件到本地
